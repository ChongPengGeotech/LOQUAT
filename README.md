# LOQUAT
GPU-accelerated SPH solver for geotechnics

1. REQUIREMENTS <br />
<br />
NVIDIA CUDA Toolkit<br />
BOOST<br />
CMAKE<br />
Ubuntu or CentOS<br />
<br />
2. Compilation

In the LOQUAT folder:
mkdir build
cd build
cmake ..
make

3. Run simulations

./LOQUAT /path/of/input/file/name_of_project
