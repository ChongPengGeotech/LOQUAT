## LOQUAT
GPU-accelerated SPH solver for geotechnics

###REQUIREMENTS###
NVIDIA CUDA Toolkit<br />
BOOST<br />
CMAKE<br />
Ubuntu or CentOS<br />

###COMPILATION###<br />
Use the following commands
```
mkdir build
cd build
cmake ..
make
```

###RUN SIMULATION###
```
./LOQUAT /path/of/input/file/name_of_project<br />
```
